# Topology

Let \\(X\\) be a non-empty set. A topology on \\(X\\) is a collection of sets \\(\tau\\) that satisfy the following:

1. \\(\emptyset , X \in \tau\\)
2. If \\(\\{U_i\\}\subset \tau\\), then \\(\cup{U_i}\in X\\)
3. If \\(U_1,...,U_n\in X}\\), then \\(\cap_{i=1}^{n}{U_i}\in X\\)
